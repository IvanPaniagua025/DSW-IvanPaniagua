Sitio web para el aprendizaje de Framework CSS y el Posicionamiento Web.

Este sitio tiene como proposito fomentar el aprendizaje de las tematicas
correspondientes al Framework CSS y el Posicionamiento Web con la finalidad
de aprender a posicionar los elementos de una pagina web y los estilos que
podemos aplicar a nuestras paginas..

Frameworks CSS

Un framework de CSS es una biblioteca de estilos genéricos
que puede ser usada para implementar diseños web. Aportan una
serie de utilidades que pueden ser aprovechadas frecuentemente
en los distintos diseños web

Wikipedia. (2021). Framework de CSS, Recuperado de: (https://es.wikipedia.org/wiki/Framework_de_CSS.).

Posicionamiento Web

Posicionamiento web, posicionamiento en buscadores o posicionamiento .
SEO se refiere a las técnicas para que una página web aparezca en las
primeras posiciones de los resultados en buscadores (como Google) para
una serie de palabras o frases.
